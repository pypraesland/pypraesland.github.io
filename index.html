Pengantar Fungsi Logika IF

Apa itu Fungsi Logika?

Fungsi di Excel yang menghasilkan nilai TRUE (Benar) atau FALSE (Salah) berdasarkan pengujian suatu kondisi.

Contoh kondisi:

  • Apakah nilai di sel A1 lebih besar dari 75?
  • Apakah teks di sel B1 adalah "Aktif"?

Mengapa Fungsi IF Penting?

Fungsi IF memungkinkan Excel untuk membuat "keputusan" secara otomatis berdasarkan data yang ada, sehingga Anda tidak perlu melakukannya secara manual.

Contoh penggunaan:

  • Menentukan status: Lulus / Gagal
  • Memberi Kategori: A / B / C
  • Memberi label: Bonus / Tidak

Dua Jenis IF Utama

1. IF Tunggal

Digunakan untuk 2 kemungkinan hasil. Logikanya sederhana, seperti "jika benar, maka ini; jika salah, maka itu".

2. IF Bertingkat (Nested IF)

Digunakan untuk lebih dari 2 kemungkinan hasil. Ini seperti menumpuk beberapa fungsi IF untuk memeriksa beberapa kondisi secara berurutan.

Menguasai keduanya adalah dasar untuk membuat formula yang lebih kompleks dan cerdas di Excel.

IF Tunggal (Simple IF)

Konsep & Sintaks

"Jika kondisi terpenuhi, lakukan A; jika tidak, lakukan B."

=IF(logical_test; value_if_true; value_if_false)
logical_test: Kondisi yang diuji (misal: A1>75). Operator yang bisa digunakan: >, <, =, >=, <=, <> (tidak sama dengan).
value_if_true: Hasil jika kondisi BENAR (misal: "Lulus"). Bisa berupa teks, angka, atau formula lain.
value_if_false: Hasil jika kondisi SALAH (misal: "Remedial").

Studi Kasus Sederhana

Masalah: Menentukan status kelulusan siswa. Kriteria: jika nilai >= 70, maka "Lulus", jika tidak, "Remedial".

Penerapan: Jika nilai siswa ada di sel D5, maka rumusnya adalah:

=IF(D5>=70; "Lulus"; "Remedial")

IF Bertingkat (Nested IF)

Konsep & Sintaks

Digunakan saat ada lebih dari dua hasil. Caranya adalah dengan memasukkan fungsi IF baru di dalam argumen value_if_false dari IF sebelumnya.

=IF(kondisi1; hasil1; IF(kondisi2; hasil2; hasil_jika_salah_semua))

Studi Kasus

Masalah: Memberi predikat nilai siswa. Kriteria: >=90 (A), >=75 (B), >=60 (C), dan sisanya (D).

Penerapan: Jika nilai siswa ada di sel D5, maka rumusnya:

=IF(D5>=90; "A"; IF(D5>=75; "B"; IF(D5>=60; "C"; "D")))

Tips Penting

Urutan Kondisi: Mulailah dari kondisi nilai tertinggi (>=90) atau terendah. Jangan mengacaknya.
Jumlah Tingkatan: Hindari lebih dari 3-4 tingkatan IF agar rumus tidak pusing dibaca. Jika lebih, gunakan alternatif lain.
Tanda Kurung: Pastikan jumlah kurung tutup `)` di akhir rumus sama dengan jumlah IF yang digunakan.

Alternatif IF Bertingkat

Fungsi IFS (Excel 2019/365 ke Atas)

Untuk kondisi bertingkat yang rumit, fungsi `IFS` jauh lebih rapi dan mudah dibaca karena tidak perlu menumpuk IF.

Sintaksnya lebih sederhana:

=IFS(kondisi1; hasil1; kondisi2; hasil2; kondisi3; hasil3; ...)

Fungsi ini adalah solusi modern untuk menggantikan IF bertingkat yang panjang dan membingungkan.

Kapan Menggunakan Masing-Masing IF?

Gunakan IF Tunggal

Ketika Anda hanya butuh dua kemungkinan hasil. Ini adalah pilihan tercepat dan paling sederhana.

  • Ya / Tidak
  • Lulus / Gagal
  • Bonus / Tidak
  • Aktif / Nonaktif

Gunakan IF Bertingkat / IFS

Ketika Anda butuh tiga atau lebih kemungkinan hasil yang didasarkan pada beberapa kriteria berurutan.

  • Predikat Nilai (A, B, C, D)
  • Kategori Produk (Premium, Standar, Ekonomi)
  • Level Diskon (10%, 5%, 0%)

Kesimpulan

Alat Fundamental

Fungsi IF adalah salah satu alat paling dasar dan kuat untuk otomatisasi pengambilan keputusan di dalam spreadsheet Anda.

Pahami Logikanya

Kunci utamanya adalah memahami tiga komponen: logical_test, value_if_true, dan value_if_false.

Pilih Sesuai Kebutuhan

Gunakan IF Tunggal untuk kasus sederhana, dan jangan ragu menggunakan IF Bertingkat atau IFS untuk kasus yang lebih kompleks.
