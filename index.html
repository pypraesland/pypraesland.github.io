Pengantar Fungsi Logika IF

? Apa itu Fungsi Logika?

Fungsi di Excel yang menghasilkan nilai TRUE (Benar) atau FALSE (Salah) berdasarkan pengujian kondisi.

Contoh kondisi:

  • Apakah nilai lebih besar dari 75?
  • Apakah statusnya "Aktif"?

! Mengapa Fungsi IF Penting?

  • Memungkinkan Excel membuat "keputusan" otomatis berdasarkan data
  • Mengisi kolom berdasarkan kriteria tertentu

Contoh penggunaan:

  • Lulus/Gagal
  • Kategori A/B/C
  • Bonus Ya/Tidak

Dasar untuk formula yang lebih kompleks dalam Excel

Dua Jenis IF Utama

1

IF Tunggal

(Simple IF)

Karakteristik:

  • Untuk 2 kemungkinan hasil
  • Satu kondisi yang diuji
  • Logika sederhana: Ya/Tidak

Contoh:

Nilai ‚â• 70 ‚Üí "Lulus"
Nilai < 70 ‚Üí "Remedial"

2

IF Bertingkat

(Nested IF)

Karakteristik:

  • Untuk lebih dari 2 kemungkinan hasil
  • Multiple kondisi berurutan
  • Logika kompleks: A/B/C/D

Contoh:

Nilai ‚â• 90 ‚Üí "A"
Nilai ‚â• 75 ‚Üí "B"
Nilai ‚â• 60 ‚Üí "C"
Nilai < 60 ‚Üí "D"

Pilihan Tergantung Kebutuhan

Gunakan IF Tunggal untuk keputusan sederhana, IF Bertingkat untuk keputusan kompleks

IF Tunggal (Simple IF)

Konsep Dasar

"Jika kondisi ini terpenuhi, lakukan A; jika tidak, lakukan B."

Hanya ada satu kondisi yang diuji, menghasilkan dua kemungkinan output.

Sintaks Rumus

=IF(logical_test; value_if_true; value_if_false)

logical_test

Kondisi yang diuji (wajib menghasilkan TRUE atau FALSE)

Contoh: A1>75, B2="Lulus", C3<=10

Operator: >, <, =, >=, <=, <> (tidak sama dengan)

value_if_true

Hasil yang ditampilkan jika logical_test BENAR

Bisa berupa teks (dalam tanda kutip, misal "Lulus"), angka, atau rumus lain

value_if_false

Hasil yang ditampilkan jika logical_test SALAH

Sama seperti value_if_true, bisa teks, angka, atau rumus

Studi Kasus Sederhana

Masalah:

Menentukan status kelulusan siswa. Jika nilai >= 70, maka "Lulus", jika tidak, "Remedial".

Contoh Penerapan:

Jika nilai ada di sel D5, rumusnya:

=IF(D5>=70; "Lulus"; "Remedial")

Contoh Hasil:

Nilai: 85‚Üí Lulus
Nilai: 65‚Üí Remedial
Nilai: 70‚Üí Lulus

IF Bertingkat (Nested IF)

Konsep Dasar

Menguji lebih dari satu kondisi secara berurutan untuk mendapatkan lebih dari dua kemungkinan hasil.

Logika: "Jika kondisi 1 benar, hasil A. Jika tidak, periksa lagi: Jika kondisi 2 benar, hasil B. Jika tidak, periksa lagi: ..., jika tidak (semua kondisi salah), hasil Z."

Fungsi IF dimasukkan ke dalam argumen value_if_false dari IF sebelumnya.

Sintaks Rumus (Contoh 3 Hasil)

=IF(logical_test1; value_if_true1; IF(logical_test2; value_if_true2; value_if_false2))

Excel akan mengevaluasi dari kiri ke kanan. Begitu satu kondisi TRUE terpenuhi, hasilnya akan ditampilkan dan evaluasi berhenti.

Studi Kasus

Masalah:

Menentukan predikat nilai: >=90 (A), >=75 (B), >=60 (C), <60 (D).

Contoh Penerapan:

Jika nilai ada di sel D5, rumusnya:

=IF(D5>=90; "A"; IF(D5>=75; "B"; IF(D5>=60; "C"; "D")))

Contoh Hasil:

Nilai: 95‚Üí A
Nilai: 80‚Üí B
Nilai: 65‚Üí C
Nilai: 50‚Üí D

Tips Penting

Urutan Kondisi

Mulailah dari kondisi yang paling spesifik/tertinggi atau terendah.

Jumlah Tingkatan

Excel modern mendukung banyak tingkatan, tetapi terlalu banyak membuat rumus sulit dibaca. Pertimbangkan alternatif jika > 3-4 tingkat.

Tanda Kurung

Pastikan jumlah tanda kurung tutup ) di akhir sesuai dengan jumlah IF yang digunakan.
