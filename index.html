Slide 2: Pengantar Fungsi Logika IF

Apa itu Fungsi Logika?

  • Fungsi di Excel yang menghasilkan nilai TRUE (Benar) atau FALSE (Salah) berdasarkan pengujian kondisi.
  • Contoh kondisi: Apakah nilai lebih besar dari 75? Apakah statusnya "Aktif"?

Mengapa Fungsi IF Penting?

  • Memungkinkan Excel membuat "keputusan" otomatis berdasarkan data.
  • Mengisi kolom berdasarkan kriteria tertentu (misal: Lulus/Gagal, Kategori A/B/C, Bonus Ya/Tidak).

Slide 3: Dasar & Jenis IF

  • Dasar untuk formula yang lebih kompleks.

Dua Jenis IF Utama:

  • IF Tunggal: Untuk 2 kemungkinan hasil.
  • IF Bertingkat (Nested IF): Untuk lebih dari 2 kemungkinan hasil.

Slide 4: IF Tunggal (Simple IF)

Konsep Dasar:

  • "Jika kondisi ini terpenuhi, lakukan A; jika tidak, lakukan B."
  • Hanya ada satu kondisi yang diuji, menghasilkan dua kemungkinan output.

Sintaks Rumus:

Excel

=IF(logical_test; value_if_true; value_if_false)

  • logical_test: Kondisi yang diuji (wajib menghasilkan TRUE atau FALSE).
    • Contoh: A1>75, B2="Lulus", C3<=10
    • Operator: >, <, =, >=, <=, <> (tidak sama dengan).
  • value_if_true: Hasil yang ditampilkan jika logical_test BENAR.
    • Bisa berupa teks (dalam tanda kutip, misal "Lulus"), angka, atau rumus lain.
  • value_if_false: Hasil yang ditampilkan jika logical_test SALAH.

Studi Kasus Sederhana:

  • Masalah: Menentukan status kelulusan siswa. Jika nilai >= 70, maka "Lulus", jika tidak, "Remedial".

Contoh Penerapan:

Jika nilai ada di sel D5, rumusnya:

=IF(D5>=70; "Lulus"; "Remedial")

Slide 5: IF Bertingkat (Nested IF)

Konsep Dasar:

  • Menguji lebih dari satu kondisi secara berurutan untuk mendapatkan lebih dari dua kemungkinan hasil.
  • Logika: "Jika kondisi 1 benar, hasil A. Jika tidak, periksa lagi: Jika kondisi 2 benar, hasil B. Jika tidak, periksa lagi: ..., jika tidak (semua kondisi salah), hasil Z."
  • Fungsi IF dimasukkan ke dalam argumen value_if_false dari IF sebelumnya.

Sintaks Rumus (Contoh 3 Hasil):

Excel

=IF(logical_test1; value_if_true1; IF(logical_test2; value_if_true2; value_if_false2))

  • Excel akan mengevaluasi dari kiri ke kanan. Begitu satu kondisi TRUE terpenuhi, hasilnya akan ditampilkan dan evaluasi berhenti.

Studi Kasus:

  • Masalah: Menentukan predikat nilai: >=90 (A), >=75 (B), >=60 (C), <60 (D).

Contoh Penerapan:

Jika nilai ada di sel D5, rumusnya:

=IF(D5>=90; "A"; IF(D5>=75; "B"; IF(D5>=60; "C"; "D")))

Tips Penting:

  • Urutan Kondisi: Mulailah dari kondisi yang paling spesifik/tertinggi atau terendah.
  • Jumlah Tingkatan: Excel modern mendukung banyak tingkatan, tetapi terlalu banyak membuat rumus sulit dibaca. Pertimbangkan alternatif jika > 3-4 tingkat.
  • Tanda Kurung: Pastikan jumlah tanda kurung tutup ) di akhir sesuai dengan jumlah IF yang digunakan.

Slide 6: Alternatif IF Bertingkat

Fungsi IFS (Excel 2019/365 ke atas):

  • Sintaks lebih rapi:

=IFS(kondisi1; hasil1; kondisi2; hasil2; kondisi3; hasil3; ...)

  • Lebih mudah dibaca untuk banyak kondisi.

Slide 7: Kapan Menggunakan Masing-Masing IF?

  • IF Tunggal: Ketika hanya ada dua kemungkinan hasil (Ya/Tidak, Lulus/Gagal, Bonus/Tidak).
  • IF Bertingkat: Ketika ada tiga atau lebih kemungkinan hasil yang bergantung pada kondisi berurutan.

Slide 8: Kesimpulan

  • Fungsi IF adalah alat fundamental untuk otomatisasi keputusan di Excel.
  • Pahami logika logical_test, value_if_true, dan value_if_false.
  • Pilih antara IF Tunggal, Bertingkat, atau IFS sesuai kompleksitas kasus.
