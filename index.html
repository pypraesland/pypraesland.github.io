üí° Pengantar Fungsi Logika IF

Apa itu Fungsi Logika?

Fungsi di Excel yang menghasilkan nilai TRUE (Benar) atau FALSE (Salah) berdasarkan pengujian sebuah kondisi.

Mengapa Penting?

Memungkinkan Excel membuat "keputusan" otomatis dan mengisi kolom berdasarkan kriteria tertentu (misal: Lulus/Gagal, Kategori A/B/C).

Dua Jenis IF Utama:

  • IF Tunggal: Untuk 2 kemungkinan hasil.
  • IF Bertingkat (Nested IF): Untuk lebih dari 2 kemungkinan hasil.

üß† Kuis Cepat: Bagian `logical_test` dari fungsi IF harus menghasilkan nilai...

‚òùÔ∏è IF Tunggal (Simple IF)

Konsep: "Jika kondisi ini terpenuhi, lakukan A; jika tidak, lakukan B."

Sintaks Rumus:

=IF(logical_test; value_if_true; value_if_false)
  • logical_test: Kondisi yang diuji (misal: A1>75).
  • value_if_true: Hasil jika kondisi BENAR (misal: "Lulus").
  • value_if_false: Hasil jika kondisi SALAH (misal: "Remedial").

üß† Kuis Kasus: Jika nilai di sel D5 adalah 80, rumus manakah yang akan menghasilkan output "Lulus"? (Kriteria Lulus >= 70)

üìö IF Bertingkat (Nested IF)

Konsep: Menguji lebih dari satu kondisi secara berurutan untuk hasil yang lebih dari dua.

Logika: "Jika kondisi 1 benar, hasil A. Jika tidak, periksa lagi: Jika kondisi 2 benar, hasil B. Jika tidak, hasil Z."

Contoh Sintaks:

=IF(D5>=90; "A"; IF(D5>=75; "B"; IF(D5>=60; "C"; "D")))

Tips Penting: Mulai dari kondisi tertinggi/terendah dan pastikan jumlah kurung tutup `)` sesuai dengan jumlah `IF`.

üß† Kuis Urutan: Dalam rumus IF bertingkat, mengapa penting menguji `nilai>=90` sebelum `nilai>=75`?

‚ÜîÔ∏è Alternatif & Kapan Menggunakannya?

Alternatif: Fungsi IFS

Untuk Excel 2019/365 ke atas, sintaksnya lebih rapi:

=IFS(kondisi1; hasil1; kondisi2; hasil2; ...)

Kapan Menggunakannya?

  • IF Tunggal: Jika hanya ada dua kemungkinan hasil (Ya/Tidak, Lulus/Gagal).
  • IF Bertingkat/IFS: Jika ada tiga atau lebih kemungkinan hasil.

üß† Kuis Skenario: Anda ingin memberi status "Bonus" jika penjualan > 100, dan "Tidak Ada Bonus" jika sebaliknya. Fungsi mana yang paling efisien?

‚úÖ Kesimpulan & Diskusi

Poin Kunci:

  • Fungsi IF adalah alat fundamental untuk otomatisasi keputusan di Excel.
  • Pahami logika logical_test, value_if_true, dan value_if_false.
  • Pilih antara IF Tunggal, Bertingkat, atau IFS sesuai kompleksitas kasus Anda.
