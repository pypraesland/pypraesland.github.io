Pengantar Fungsi Logika IF

?Apa itu Fungsi Logika?

Fungsi di Excel yang menghasilkan nilai TRUE (Benar) atau FALSE (Salah) berdasarkan pengujian kondisi.

Contoh kondisi:

  • Apakah nilai lebih besar dari 75?
  • Apakah statusnya "Aktif"?

!Mengapa Fungsi IF Penting?

  • Memungkinkan Excel membuat "keputusan" otomatis berdasarkan data
  • Mengisi kolom berdasarkan kriteria tertentu

Contoh penggunaan:

  • Lulus/Gagal
  • Kategori A/B/C
  • Bonus Ya/Tidak

Dasar untuk formula yang lebih kompleks dalam Excel

Dua Jenis IF Utama

1

IF Tunggal

(Simple IF)

Karakteristik:

  • Untuk 2 kemungkinan hasil
  • Satu kondisi yang diuji
  • Logika sederhana: Ya/Tidak

Contoh:

Nilai ≥ 70 → "Lulus"
Nilai < 70 → "Remedial"

2

IF Bertingkat

(Nested IF)

Karakteristik:

  • Untuk lebih dari 2 kemungkinan hasil
  • Multiple kondisi berurutan
  • Logika kompleks: A/B/C/D

Contoh:

Nilai ≥ 90 → "A"
Nilai ≥ 75 → "B"
Nilai ≥ 60 → "C"
Nilai < 60 → "D"

Pilihan Tergantung Kebutuhan

Gunakan IF Tunggal untuk keputusan sederhana, IF Bertingkat untuk keputusan kompleks

IF Tunggal (Simple IF)

Konsep Dasar

"Jika kondisi ini terpenuhi, lakukan A; jika tidak, lakukan B."

Hanya ada satu kondisi yang diuji, menghasilkan dua kemungkinan output.

Sintaks Rumus

=IF(logical_test; value_if_true; value_if_false)

logical_test

Kondisi yang diuji (wajib menghasilkan TRUE atau FALSE)

Contoh: A1>75, B2="Lulus", C3<=10

Operator: >, <, =, >=, <=, <> (tidak sama dengan)

value_if_true

Hasil yang ditampilkan jika logical_test BENAR

Bisa berupa teks (dalam tanda kutip, misal "Lulus"), angka, atau rumus lain

value_if_false

Hasil yang ditampilkan jika logical_test SALAH

Sama seperti value_if_true, bisa teks, angka, atau rumus

Studi Kasus Sederhana

Masalah:

Menentukan status kelulusan siswa. Jika nilai >= 70, maka "Lulus", jika tidak, "Remedial".

Contoh Penerapan:

Jika nilai ada di sel D5, rumusnya:

=IF(D5>=70; "Lulus"; "Remedial")

Contoh Hasil:

Nilai: 85→ Lulus
Nilai: 65→ Remedial
Nilai: 70→ Lulus

IF Bertingkat (Nested IF)

Konsep Dasar

Menguji lebih dari satu kondisi secara berurutan untuk mendapatkan lebih dari dua kemungkinan hasil.

Logika: "Jika kondisi 1 benar, hasil A. Jika tidak, periksa lagi: Jika kondisi 2 benar, hasil B. Jika tidak, periksa lagi: ..., jika tidak (semua kondisi salah), hasil Z."

Fungsi IF dimasukkan ke dalam argumen value_if_false dari IF sebelumnya.

Sintaks Rumus (Contoh 3 Hasil)

=IF(logical_test1; value_if_true1; IF(logical_test2; value_if_true2; value_if_false2))

Excel akan mengevaluasi dari kiri ke kanan. Begitu satu kondisi TRUE terpenuhi, hasilnya akan ditampilkan dan evaluasi berhenti.

Studi Kasus

Masalah:

Menentukan predikat nilai: >=90 (A), >=75 (B), >=60 (C), <60 (D).

Contoh Penerapan:

Jika nilai ada di sel D5, rumusnya:

=IF(D5>=90; "A"; IF(D5>=75; "B"; IF(D5>=60; "C"; "D")))

Contoh Hasil:

Nilai: 95→ A
Nilai: 80→ B
Nilai: 65→ C
Nilai: 50→ D

Tips Penting

Urutan Kondisi

Mulailah dari kondisi yang paling spesifik/tertinggi atau terendah.

Jumlah Tingkatan

Excel modern mendukung banyak tingkatan, tetapi terlalu banyak membuat rumus sulit dibaca. Pertimbangkan alternatif jika > 3-4 tingkat.

Tanda Kurung

Pastikan jumlah tanda kurung tutup ) di akhir sesuai dengan jumlah IF yang digunakan.

Alternatif IF Bertingkat

IFS

Fungsi IFS

(Excel 2019/365 ke atas)

Sintaks Lebih Rapi

=IFS(kondisi1; hasil1; kondisi2; hasil2; kondisi3; hasil3; ...)
  • Tidak perlu nested IF
  • Lebih mudah dibaca
  • Lebih mudah di-maintain

Contoh Perbandingan

Nested IF (Rumit):

=IF(D5>=90;"A";IF(D5>=75;"B";IF(D5>=60;"C";"D")))

IFS (Lebih Rapi):

=IFS(D5>=90;"A";D5>=75;"B";D5>=60;"C";TRUE;"D")

Keunggulan Fungsi IFS

Lebih Mudah Dibaca

Struktur yang lebih jelas dan terorganisir

Lebih Mudah Maintenance

Mudah menambah atau mengubah kondisi

Performa Lebih Baik

Evaluasi yang lebih efisien

Catatan Penting

Fungsi IFS hanya tersedia di Excel 2019, Excel 365, dan versi yang lebih baru. Jika menggunakan Excel versi lama, tetap gunakan Nested IF.

Kapan Menggunakan Masing-Masing IF?

Panduan Pemilihan

Berapa kemungkinan hasil yang dibutuhkan?
2

IF Tunggal

Ketika hanya ada dua kemungkinan hasil

Contoh: Ya/Tidak
Contoh: Lulus/Gagal
Contoh: Bonus/Tidak
3+

IF Bertingkat

Ketika ada tiga atau lebih kemungkinan hasil yang bergantung pada kondisi berurutan

Contoh: Grade A/B/C/D
Contoh: Kategori Tinggi/Sedang/Rendah
Contoh: Status VIP/Premium/Regular

Contoh Praktis dalam Kehidupan Sehari-hari

2Gunakan IF Tunggal

Absensi Karyawan

Jam masuk <= 08:00 → "Tepat Waktu", selainnya → "Terlambat"

Status Pembayaran

Tanggal bayar <= Due Date → "Lunas", selainnya → "Overdue"

Kelayakan Kredit

Gaji >= 5 juta → "Layak", selainnya → "Tidak Layak"

3+Gunakan IF Bertingkat

Sistem Penilaian

90-100: A, 75-89: B, 60-74: C, <60: D

Kategori Usia

0-12: Anak, 13-17: Remaja, 18-59: Dewasa, 60+: Lansia

Level Diskon

Pembelian >1jt: 15%, >500rb: 10%, >100rb: 5%, lainnya: 0%

Tips Pemilihan yang Tepat

  • Mulai dengan menghitung berapa kemungkinan hasil yang dibutuhkan
  • Jika hanya 2 hasil → pilih IF Tunggal untuk kesederhanaan
  • Jika 3+ hasil → pertimbangkan IF Bertingkat atau IFS (jika tersedia)
  • Untuk kondisi yang sangat kompleks (>5 tingkat), pertimbangkan menggunakan tabel lookup atau fungsi lain

Kesimpulan

Alat Fundamental

Fungsi IF adalah alat fundamental untuk otomatisasi keputusan di Excel

Pahami Logika

Pahami logika logical_test, value_if_true, dan value_if_false

Pilih Sesuai Kebutuhan

Pilih antara IF Tunggal, Bertingkat, atau IFS sesuai kompleksitas kasus
